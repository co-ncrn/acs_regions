Regionalization Results

We created four scenarios, Housing, Poverty, Transportation, and a General scenario, each containing a different combination of input variables from the 2008-2012 Five-Year American Community Survey. We then ran the regionalization code for each scenario for every Metropolitan Statistical Area in the United States (as designated in 2013). The downloadable results include values for the input variables at the tract level, values of the output variables at the region level, a crosswalk between tracts and regions, shapefiles for the input tracts and output regions, and GeoJSON maps containing the data for the tracts and regions, a data dictionary for the original and derived variables, and a metadata file.
